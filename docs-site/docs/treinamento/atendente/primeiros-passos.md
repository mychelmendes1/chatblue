# Primeiros Passos no ChatBlue

Bem-vindo ao ChatBlue! Este guia vai te ajudar a comecar a atender clientes via WhatsApp de forma profissional e eficiente.

## O Que e o ChatBlue?

O ChatBlue e uma plataforma de atendimento ao cliente que conecta voce aos clientes atraves do WhatsApp. Com ele, voce pode:

- Atender multiplos clientes simultaneamente
- Contar com ajuda de Inteligencia Artificial
- Transferir atendimentos para outros colegas
- Acessar o historico completo de conversas
- Usar respostas rapidas pre-configuradas

---

## Fazendo Login

### Passo 1: Acesse o Sistema

1. Abra seu navegador (recomendamos Chrome ou Firefox)
2. Digite o endereco do sistema fornecido pelo seu administrador
3. Voce vera a tela de login

![Tela de Login](/img/treinamento/login.png)

### Passo 2: Entre com suas Credenciais

1. Digite seu **email** corporativo
2. Digite sua **senha** (fornecida pelo administrador)
3. Clique em **Entrar**

:::tip Primeiro Acesso
No primeiro acesso, recomendamos trocar sua senha. Va em Configuracoes > Minha Conta > Alterar Senha.
:::

### Passo 3: Selecione sua Empresa (se aplicavel)

Se voce tem acesso a mais de uma empresa, selecione a empresa correta na lista.

## Entendendo os Status

### Status das Conversas

| Cor | Status | Significado |
|-----|--------|-------------|
| 🟡 | **Pendente** | Aguardando primeiro atendimento |
| 🔵 | **Em Atendimento** | Voce esta atendendo |
| 🟠 | **Aguardando** | Esperando resposta do cliente |
| 🟢 | **Resolvido** | Atendimento concluido |
| ⚫ | **Fechado** | Ticket encerrado |

### Status das Mensagens

| Icone | Significado |
|-------|-------------|
| 🕐 | Enviando... |
| ✓ | Enviada ao servidor |
| ✓✓ | Entregue ao cliente |
| ✓✓ (azul) | Lida pelo cliente |

## Seu Primeiro Atendimento

### Passo 1: Pegue um Ticket da Fila

1. Clique na aba **"Fila"** na lista de conversas
2. Voce vera os tickets aguardando atendimento
3. Clique em um ticket para visualizar
4. Clique no botao **"Assumir Atendimento"**

:::info Ticket Atribuido
Agora este ticket e seu! O cliente vera que esta sendo atendido por voce.
:::

### Passo 2: Leia o Historico

Antes de responder:
1. Leia todas as mensagens anteriores
2. Verifique se a IA ja respondeu algo
3. Entenda o que o cliente precisa

### Passo 3: Responda o Cliente

1. Digite sua mensagem no campo de texto
2. Pressione **Enter** ou clique no botao de enviar
3. Aguarde a confirmacao de envio (✓)

### Passo 4: Resolva ou Transfira

Apos resolver o problema:
1. Clique em **"Resolver Ticket"** se o atendimento foi concluido
2. Ou clique em **"Transferir"** se outro departamento deve continuar

## Atalhos de Teclado

Aprenda estes atalhos para ser mais rapido:

| Atalho | Acao |
|--------|------|
| `Enter` | Enviar mensagem |
| `Shift + Enter` | Quebra de linha (nova linha) |
| `Ctrl + K` | Buscar conversas |
| `Ctrl + /` | Ver todos os atalhos |
| `Esc` | Fechar painel lateral |

## Dicas para o Primeiro Dia

:::tip Dica 1: Nao Tenha Pressa
E normal demorar mais no comeco. A velocidade vem com a pratica!
:::

:::tip Dica 2: Peca Ajuda
Se nao souber responder algo, pergunte ao supervisor ou transfira para quem sabe.
:::

:::tip Dica 3: Personalize
Use o nome do cliente e seja cordial. Isso faz diferenca!
:::

:::tip Dica 4: Revise Antes de Enviar
Releia sua mensagem antes de enviar. Erros de digitacao passam uma imagem ruim.
:::
