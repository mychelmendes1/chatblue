# Centro de Treinamento ChatBlue

Bem-vindo ao centro de treinamento do ChatBlue! Esta secao foi criada para ajudar voce a dominar todas as funcionalidades do sistema de atendimento.

## Para Quem e Este Treinamento?

Este material foi desenvolvido para tres perfis de usuarios:

### Atendentes
Voce e responsavel por atender os clientes via WhatsApp. Aqui voce aprendera:
- Como usar a interface de chat
- Gerenciar suas conversas
- Transferir atendimentos
- Usar respostas rapidas
- Trabalhar junto com a IA

**[Ir para Manual do Atendente →](/treinamento/atendente/primeiros-passos)**

### Supervisores
Voce gerencia uma equipe de atendentes. Aqui voce aprendera:
- Monitorar a equipe em tempo real
- Analisar metricas e SLA
- Redistribuir atendimentos
- Gerar relatorios
- Treinar novos atendentes

**[Ir para Manual do Supervisor →](/treinamento/supervisor/visao-geral)**

### Administradores
Voce configura e gerencia o sistema. Aqui voce aprendera:
- Configurar a Inteligencia Artificial
- Gerenciar usuarios e departamentos
- Configurar conexoes WhatsApp
- Personalizar o sistema
- Integrar com outros sistemas

**[Ir para Manual do Administrador →](/treinamento/administrador/painel-controle)**

## Precisa de Ajuda?

Se voce encontrar dificuldades durante o treinamento:

1. **Consulte este manual** - A maioria das duvidas esta respondida aqui
2. **Pergunte ao seu supervisor** - Eles estao preparados para ajudar
3. **Use o chat interno** - Converse com colegas mais experientes
4. **Reporte problemas** - Se encontrar bugs, avise o administrador
