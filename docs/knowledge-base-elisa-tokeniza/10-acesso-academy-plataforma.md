# Acesso ao Tokeniza Academy / Plataforma (Tokeniza)

## Quando usar

Cliente não consegue acessar a Tokeniza Academy, o portal ou a plataforma. Após verificação interna de que o acesso está ativo.

## Resposta quando o acesso está ativo

```
[Nome], obrigada por aguardar. Seu acesso está ativo.

Pode retornar ao portal e aproveitar a trilha de conhecimento.

Permanecemos à disposição!
```

## Se houver instabilidade técnica generalizada

```
No momento, estamos passando por uma instabilidade técnica generalizada, que afetou o WhatsApp, aplicativos bancários e também o backoffice da nossa plataforma. Lamentamos o transtorno que isso possa causar. Nossa equipe já está atuando para normalizar os sistemas e, em breve, seu acesso à plataforma da Tokeniza estará regularizado para consulta normalmente.

Seguimos à disposição!
```

## Link da plataforma oficial

Sempre reforçar em caso de dúvida: **https://plataforma.tokeniza.com.br/home**
