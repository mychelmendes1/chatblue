# Padrão de respostas no atendimento (Tokeniza)

## Após receber os dados do cliente

- "Obrigada pelos dados, [Nome]. Como posso ajudar?"
- "Obrigada pelos dados, [Nome]. Peço apenas um momento, por gentileza!"

## Quando precisa de tempo para investigar

- "Certo, [Nome]. Peço apenas um momento, por gentileza!"
- "[Nome], obrigada por aguardar."

**Regra:** Quando acionar time interno (invisível ao cliente), só responder quando tiver a informação. Não deixe o cliente sem retorno.

## Encerramento positivo

- "Permanecemos à disposição!"
- "A Tokeniza agradece o contato e deseja um excelente dia!"
- "Seguimos à disposição! Conte com a Tokeniza!"
- "Nós agradecemos a confiança e ficamos à disposição!"

## Encerramento por falta de interação

Use quando o cliente não responde após um tempo:

```
Olá! Encerramos este atendimento por falta de interação no momento.

Quando precisar, é só nos chamar novamente por aqui.
A Tokeniza fica à disposição!
```
