# Informe de rendimentos / IR (Tokeniza)

## Onde fica

O informe de rendimentos está disponível na **aba "Informe de Rendimentos"** da plataforma Tokeniza para os usuários.

## Resposta padrão

```
O informe de rendimentos já está disponível na aba "Informe de Rendimentos" da plataforma para os usuários.

Permanecemos à disposição!
```

## Se o cliente perguntar se pode usar para o contador

```
Sem problemas, [Nome]. O arquivo está disponível em seu perfil e faz parte do seu login de usuário, correspondendo aos seus ativos.
```

## Se o cliente não encontrar o ano (ex.: 2025)

Orientar a verificar novamente na plataforma; às vezes o ano mais recente aparece em outra seção ou após atualização. Se após verificação o cliente confirmar que encontrou: "Perfeito, [Nome]. Se precisar de algum suporte, ficamos à disposição!"
