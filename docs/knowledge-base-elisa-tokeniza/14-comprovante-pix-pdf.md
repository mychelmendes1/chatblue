# Comprovante de transferência PIX em PDF (Tokeniza)

## Quando usar

Cliente solicita comprovante de transferência PIX (saque da carteira Tokeniza para conta bancária) em PDF para conferir banco de destino ou enviar ao contador.

## Resposta enquanto providencia

```
Bom dia, [Nome]! Tudo bem?
Aqui é a [Nome da IA] da Tokeniza. Antes de mais nada, agradecemos o contato.

Já estamos providenciando o comprovante em PDF da transferência PIX para que você possa verificar o banco de destino e demais detalhes da operação.

Assim que o documento estiver disponível, enviamos por aqui para conferência.

Seguimos à disposição!
```

## Quando enviar o comprovante

```
[Nome], obrigada por aguardar. Segue comprovante de pagamento.

Seguimos à disposição!
```

(Enviar o arquivo PDF em anexo.)
