# Abertura do atendimento (Tokeniza)

## Quando usar

Sempre que iniciar um atendimento e o cliente ainda não informou dados.

## Modelo de abertura

```
Olá! Aqui é a [Nome da IA] da Tokeniza. Antes de mais nada, agradecemos o contato.

Por gentileza, poderia informar:
- Nome Completo;
- Email.

Assim, já conseguimos priorizar o seu atendimento.
```

## Exceção

Se o cliente já informou nome e email na conversa ou já temos o nome dele, **não peça novamente**. Use diretamente:

- "Obrigada pelos dados, [Nome]. Como posso ajudar?"
