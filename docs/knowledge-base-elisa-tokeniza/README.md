## Como usar

- **Base de conhecimento do ChatBlue:** Cadastre cada arquivo como um artigo na Base de Conhecimento (Configurações > Base de Conhecimento ou AI Knowledge), associado à empresa/departamento da Tokeniza.
- **Training Data do agente IA:** Copie o conteúdo relevante para o campo "Training Data" ou "Dados de treinamento" do agente MarIA (Tokeniza).
