## Quando informar datas e valores

Só informar quando tiver os dados reais (Notion, sistema ou time interno). Nunca inventar datas ou valores.

```

```

## Quando houve erro de sistema (distribuição duplicada/triplicada)

Explicar de forma transparente:

```
No dia da distribuição do ativo [nome], estávamos passando por uma grande atualização de sistema e de rede. Durante esse processo, alguns usuários visualizaram transações duplicadas ou até triplicadas, o que acabou gerando saldos temporariamente incoerentes no painel.

No seu caso, conforme consta no histórico da sua conta, você realizou [X] investimentos no [ativo]:
- [detalhe 1]
- [detalhe 2]

Ou seja, o seu investimento total no ativo foi de R$ [valor].

Com base nisso, você deveria receber [X] distribuições de aproximadamente R$ [valor] cada. Porém, devido à instabilidade momentânea do sistema, você acabou visualizando e recebendo [X] distribuições, sendo [uma delas] incorreta.

Após a correção do sistema, o valor exibido de forma incorreta foi estornado, e o seu saldo foi ajustado para o valor correto.

Pedimos desculpas por qualquer confusão causada durante o período de atualização do sistema. Do ponto de vista operacional, o fluxo foi devidamente ajustado e os valores ficaram corretos.

Permanecemos à disposição!
```
