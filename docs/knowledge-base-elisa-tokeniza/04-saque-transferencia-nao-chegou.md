# Saque / Transferência não chegou (Tokeniza)

## Contexto

Cliente fez saque de tBRL, BTC, ATH ou outro ativo e o valor não apareceu no destino. Verificar a transação na plataforma e na blockchain antes de responder.

## Resposta quando a transação foi efetivada

```
De acordo com nossa plataforma, a transação foi efetivada.

A mesma pode ser identificada na blockchain:
[inserir link da transação no etherscan ou explorer correspondente]

Permanecemos à disposição!
```

## Valor baixo em cripto (ex.: ATH) enviado para exchange

Muitas exchanges têm valor mínimo (threshold) para crédito. Orientar:

```
Um ponto importante: caso o envio tenha sido feito para uma exchange, muitas plataformas possuem valor mínimo (threshold) para que o depósito seja reconhecido ou exibido no saldo.

No seu caso, o valor enviado foi de [valor], que atualmente é um montante bastante baixo e pode não atingir o mínimo exigido pela exchange para crédito automático na conta.

👉 Nessa situação, orientamos que você entre em contato diretamente com:
- O suporte da exchange de destino (caso tenha enviado para corretora), ou
- O provedor da carteira utilizada no recebimento, para verificar as regras de depósito mínimo e eventuais políticas internas de exibição de saldo.

Infelizmente, após a confirmação on-chain, não há como a Tokeniza intervir no processamento interno da exchange ou da carteira de destino.

Seguimos à disposição!
```

## Regra

Nunca inventar link de blockchain. Sempre verificar a transação no sistema e incluir o link real da comprovação on-chain.
